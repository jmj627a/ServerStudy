# dib 출력 샘플 코드

DIB (bmp 파일)  을 로딩하여  GDI 로 출력하는 샘플 코드를 첨부 합니다.
문서를 읽어보시고,  첨부된 소스코드를 사용해서 bmp 파일을 로딩하여 화면에 출력하는 내용을 진행 해보시기 바랍니다.

GDI bmp 출력 함수를 사용하는 경우 해당 DIB 의 색상에 따른 변환 등의 작업이 필요가 없습니다.
소스파일과 bmp 파일만 첨부 하오니 프로젝트를 만들어서 샘플 코드를 돌려 보시기 바랍니다.


WM_PAINT 발생시 

bmp 오픈 -> 로딩 -> 메모리에 dib 준비 -> StretchDIBits 로 출력  

의 절차로 출력이 되는 샘플 입니다.

간단한 출력이 이해 되셨다면  해당 bmp 의 이미지를  변경 ( 조금 어둡게, 그레이스케일, 조금밝게 등... )  처리하는 코드도 만들어 보시기 바랍니다.
